Command line options

You can get the list of supported scorche3d options by running scorched3d with /? or --help on the command line.
The following are the current command line options for Scorched Earth 3D: -
FlagDescription
-allowexceptionsAllows any program exceptions to be thrown.
-connect [name] Starts a NET/LAN client by connecting to NAME.
-password Password needed for NET/LAN client to connect.
-startserver [filename] Starts a dedicated Scorched 3D Server. Specify server file [data/server.xml]
-startclient [filename] Starts the single player version of Scorched3D. Specify client file [data/singleeasy.xml]
