Aiming/Firing Keys

These keys allow the user to change the aim of the current tank. See here for more information on aiming.
Note: When aiming or changing power the shift key will slow down the changes, the control key will speed up the changes and the shif and control key together will make the changes really slow.
Key Action
Arrow Keys/Cursor Keys (not keypad) Rotate and elevate the aim of the current tank.
+/- Keys (Plus/Minus) Increase and decrease the power that will be used for the next shot.
A key Aim at the position on the landscape pointed to by the mouse pointer. This only sets the tanks rotation NOT the elevation or power.
Space Bar Fire the currently selected weapon using the tanks currently set direction and power.
Tab Key Cycle through the set of available weapons and select the next available. Pressing the left shift key at the same time will reverse the cycling order. The number of available weapons depends on the weapons the player has bought.
U Key Reset the rotation, elevation and power back to those used for the last shot.

Quitting

Key Action
Esc Key Quit game.
Q Key Kill all players.

Misc Keys

Key Action
I Key Toggle HUD displays on or off.
T Key NET/LAN chat/talk.
Y Key NET/LAN team chat/talk.
K Key Say "No Kibitzing!!".
S Key Show the score view.
R Key Show the game rules.
M Key Show the player name dialog.
F1->F4 Keys Increase game speed 1X -> 4X (Single player only).
F5 Key Take screenshot.
` Key Show the console. Type help in console for list of commands.

Accessory Keys

Key Action
P Key Toggle Parachutes on or off.
B Key Use one battery.

Camera Keys

The camera keys allows the current camera position to be changed. See here for more information on the camera.
Key Action
Keyboard number keys 1->9 Change to a pre-defined camera angle/preset.
Hold control and the Keyboard number keys 1->5 Store a new user defined camera preset
Hold shift and the Keyboard number keys 1->5 Change to a user defined camera preset
Left Alt Key Allow the camera view to penetrate the landscape.
C Key Turn on the secondary camera view.

Customizing Keys

All keys can be customized by editing the keys.xml file found in the Scorched3D data directory.
